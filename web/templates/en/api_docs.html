API Documentation

Integrate JobHunt Pro into your workflow. All API requests require authentication via your API key.

🔑

Your API Key

Find your API key in the Wallet section of your dashboard. Include it in the X-API-Key header on all requests.

X-API-Key: {{ api_key if api_key else 'jhp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx' }}

Endpoints

GET /api/v1/health Health Check
Returns the current health status of the platform. No authentication required.
{
  "status": "healthy",
  "timestamp": "2026-06-05T08:44:00Z"
}
GET /api/v1/profile Get Profile
Retrieve your account profile including CV data, skills, and preferences.

Headers

NameTypeDescription
X-API-Keystringrequired Your API key
{
  "user_id": "user_...",
  "email": "...",
  "name": "...",
  "cv_profiles": [...]
}
GET /api/v1/campaigns List Campaigns
Retrieve all your campaigns with status and statistics.

Headers

NameTypeDescription
X-API-Keystringrequired Your API key

Query Parameters

NameTypeDefaultDescription
limitinteger20Max campaigns to return
statusstring—Filter by status (pending, running, completed, failed)
{
  "campaigns": [
    {
      "campaign_id": "camp_...",
      "status": "completed",
      "total_companies": 50,
      "sent_count": 48,
      "response_count": 3
    }
  ]
}
POST /api/v1/campaigns Create Campaign
Create and launch a new job application campaign.

Headers

NameTypeDescription
X-API-Keystringrequired Your API key
Content-Typestringrequired application/json

Body Parameters

NameTypeDescription
profile_idintegerrequired CV profile ID to use
querystringrequired Job search query (e.g. "network engineer")
locationstringTarget location (e.g. "Dubai, UAE")
company_countintegerrequired Number of companies to target
curl -X POST https://jhfguf.pythonanywhere.com/api/v1/campaigns \
  -H "X-API-Key: jhp_..." \
  -H "Content-Type: application/json" \
  -d '{"profile_id": 1, "query": "network engineer", "location": "Dubai, UAE", "company_count": 50}'
GET /api/v1/stats Application Stats
Get aggregated statistics about your applications.

Headers

NameTypeDescription
X-API-Keystringrequired Your API key
{
  "applied": 142,
  "viewed": 48,
  "responded": 12,
  "interview": 4,
  "offer": 1
}
GET /api/v1/pipeline Pipeline Data
Get your application pipeline with stage breakdown.

Headers

NameTypeDescription
