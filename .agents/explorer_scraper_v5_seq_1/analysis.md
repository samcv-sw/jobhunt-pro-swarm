# Scraper Stealth & Proxy Hardening Analysis

## 1. Executive Summary
This document presents a read-only investigation and design of scraper stealth and proxy hardening requirements for JobHunt Pro. Specifically, it reviews the logic in `core/stealth.py` and `scrapers/stealth_ingest.py`, tracks how proxies are passed to browser automation fallbacks (`NodriverFallback` and `ApexCamoufoxFallback`), details where and how real IP leaks could occur, verifies returned scraper data structures, and outlines precise recommended modifications along with test verification plans.

---

## 2. Proxy Passing & IP Leak Analysis

### Current Implementation & Flow
1. **Initial Proxy Config Generation**:
   - In `scrapers/stealth_ingest.py`, a proxy dictionary `proxy_config` is generated by calling `get_stabilized_proxy(session_id)` (lines 409-410).
   - This function pulls proxy strings from the global variable `PROXY_LIST`, which is parsed from the environment variable `RESIDENTIAL_PROXIES` or defaults to `"http://jobhunt-stub-proxy:8080"` (line 21).
   - `proxy_config` returned is of the form `{"http": proxy_str, "https": proxy_str}` (or `{}` if no proxies are defined).

2. **Passing to Direct Request Session**:
   - The direct request session (via `curl_cffi` / `requests.AsyncSession`) receives `proxy_config` directly via its `proxies` parameter (lines 428-433).

3. **Passing to Browser Automation Fallbacks**:
   - If the direct session is challenged or fails, `process_single_job` initiates fallbacks to `NodriverFallback` and `ApexCamoufoxFallback` (lines 459-476).
   - The code extracts `proxy_str = proxy_config.get("http") if proxy_config else None` (lines 463, 472).
   - This single string is passed to:
     - `NodriverFallback.get_page_content(url, proxy=proxy_str)`
     - `ApexCamoufoxFallback.get_page_content(url, proxy=proxy_str)`

### Vulnerability Points & IP Leak Scenarios
A real IP leak (revealing the scraper's hosting machine IP to target servers) can occur at several points if the proxy configurations are missing, invalid, or ignored:

1. **Fallback Default to `None`**:
   - In `core/stealth.py`:
     - `NodriverFallback.get_page_content` constructs Chrome arguments:
       ```python
       browser_args = []
       if proxy:
           browser_args.append(f"--proxy-server={proxy}")
       ```
       If `proxy` is `None` or an empty string, **no proxy argument is added to Chrome**. Chromedriver launches Chrome and routes traffic directly through the host network interface, causing a **critical real IP leak**.
     - `ApexCamoufoxFallback.get_page_content` instantiates:
       ```python
       async with AsyncCamoufox(headless=True, proxy=proxy) as browser:
       ```
       Similarly, if `proxy` is `None`, `AsyncCamoufox` executes queries without any proxy layer, resulting in a **real IP leak**.

2. **Missing Env Var Defaulting Behavior**:
   - In `scrapers/stealth_ingest.py`, `PROXY_LIST` defaults to `["http://jobhunt-stub-proxy:8080"]` if `RESIDENTIAL_PROXIES` is not set.
   - However, if `RESIDENTIAL_PROXIES` is set to an empty string (e.g. `RESIDENTIAL_PROXIES=""` in production config/env files), `PROXY_LIST` is evaluated as an empty list `[]`.
   - If `PROXY_LIST` is empty, `get_stabilized_proxy` returns `{}`.
   - In `process_single_job`, `proxy_config` becomes `{}`. Consequently, `proxy_str` extracted from `proxy_config.get("http")` evaluates to `None`.
   - The execution flows into `NodriverFallback` and `ApexCamoufoxFallback` with `proxy=None`. Traffic is routed directly from the host system, leaking the real IP.

3. **Fallback Injection Logic inside `core/stealth.py`**:
   - The fallback classes inside `core/stealth.py` do not perform self-healing or validation on the proxy input. If they receive `None` or an empty string, they do not attempt to read the active fallback proxy (like `RESIDENTIAL_PROXIES` or `http://jobhunt-stub-proxy:8080`). They proceed directly with the unproxied browser request.

## 3. Scraper Structured Return Check

We reviewed the parsing routines in `scrapers/stealth_ingest.py` to verify that all scraper blocks yield a structured `list[dict]` containing at minimum `title` and `url` keys:

### 1. `_parse_job_page(html, source_url)`
- Returns a single dictionary:
  ```python
  return {
      "title": title,
      "url": source_url,
      "company": company,
      "description_snippet": description_snippet,
  }
  ```
- Guaranteed to contain keys `"title"` (defaults to `"Unknown Position"`) and `"url"` (defaults to `source_url`).

### 2. `_extract_jobs_from_dict(data, source_url)`
- Extracts a list of structured dictionaries:
  ```python
  results.append({
      "title": title,
      "url": url,
      "company": company,
      "description_snippet": desc[:500]
  })
  ```
- Guaranteed to contain keys `"title"` and `"url"`.

### 3. `_parse_json_ld(html, source_url)`
- Calls `_extract_jobs_from_dict(data, source_url)` recursively and returns a flat list of dictionaries.

### 4. `_parse_page_content(html, source_url)`
- **JSON-LD Branch**: Returns `_parse_json_ld(html, source_url)`, which is a `list[dict]`.
- **Card-based Parser Branch**: Iterates over matching cards and appends elements to `results`:
  ```python
  results.append({
      "title": title,
      "url": job_url,
      "company": company,
      "description_snippet": description_snippet
  })
  ```
  Guaranteed to contain keys `"title"` and `"url"`.
- **Single Page Fallback Branch**: Invokes `_parse_job_page` and wraps it in a list: `[single]`.
- **Empty / Failure Cases**: Returns `[]`.

### 5. `_parse_html_with_llm(html, source_url)`
- Sanitizes the HTML and parses it using an LLM JSON structure. Returns:
  ```python
  cleaned_jobs.append({
      "title": job.get("title") or "Unknown Position",
      "url": job.get("url") or source_url,
      "company": job.get("company"),
      "description_snippet": job.get("description_snippet", "")
  })
  ```
- Guaranteed to return a `list[dict]` containing `"title"` and `"url"`.

### 6. `process_single_job(url, session_id)`
- Fetches HTML and invokes `_parse_page_content` and (if needed) `_parse_html_with_llm`.
- Explicitly enforces constraints by cleaning returned lists:
  ```python
  cleaned_jobs = []
  for job in jobs:
      if isinstance(job, dict):
          cleaned_job = {
              "title": job.get("title") or "Unknown Position",
              "url": job.get("url") or url,
              "company": job.get("company"),
              "description_snippet": job.get("description_snippet", "")
          }
          cleaned_jobs.append(cleaned_job)
  return cleaned_jobs
  ```
- Thus, `process_single_job` is guaranteed to return `list[dict]` containing at minimum the `"title"` and `"url"` keys.

### 7. `stealth_scrape_jobs(urls)`
- Gather results across all processes. Flat-maps the lists:
  ```python
  results = []
  for r in results_raw:
      ...
      if isinstance(r, list):
          for item in r:
              if isinstance(item, dict):
                  results.append({
                      "title": item.get("title") or "Unknown Position",
                      "url": item.get("url") or "Unknown URL",
                      "company": item.get("company"),
                      "description_snippet": item.get("description_snippet", "")
                  })
  ```
- This ensures output integrity and consistency.

## 4. Draft of Proposed Changes

To guarantee zero IP leakage and robust proxy enforcement, the following changes are drafted for consideration. *No files have been modified.*

### A. Changes to `core/stealth.py`
We should update both `NodriverFallback.get_page_content` and `ApexCamoufoxFallback.get_page_content` to resolve a proxy from environment variables or fall back to the enterprise stub proxy if the passed parameter is missing, empty, or `None`.

#### 1. In `NodriverFallback.get_page_content`:
**Before:**
```python
    @staticmethod
    async def get_page_content(url: str, proxy: Optional[str] = None, timeout_seconds: int = 20) -> str:
        try:
            import nodriver as uc
            import asyncio

            browser_args = []
            if proxy:
                browser_args.append(f"--proxy-server={proxy}")
```

**Proposed After:**
```python
    @staticmethod
    async def get_page_content(url: str, proxy: Optional[str] = None, timeout_seconds: int = 20) -> str:
        try:
            import nodriver as uc
            import asyncio
            import os

            # Hardened proxy lookup to prevent real IP leaks
            if not proxy:
                proxies_env = os.getenv("RESIDENTIAL_PROXIES", "").split(",")
                valid_proxies = [p.strip() for p in proxies_env if p.strip()]
                if valid_proxies:
                    proxy = valid_proxies[0]
                else:
                    proxy = "http://jobhunt-stub-proxy:8080"

            browser_args = []
            if proxy:
                browser_args.append(f"--proxy-server={proxy}")
```

#### 2. In `ApexCamoufoxFallback.get_page_content`:
**Before:**
```python
    @staticmethod
    async def get_page_content(url: str, proxy: str = None) -> str:
        try:
            from camoufox.async_api import AsyncCamoufox
            import asyncio
            from core.human_mouse import HumanMouse

            # Using Camoufox which intercepts and overrides WebGL/Canvas natively
            async with AsyncCamoufox(headless=True, proxy=proxy) as browser:
```

**Proposed After:**
```python
    @staticmethod
    async def get_page_content(url: str, proxy: str = None) -> str:
        try:
            from camoufox.async_api import AsyncCamoufox
            import asyncio
            import os
            from core.human_mouse import HumanMouse

            # Hardened proxy lookup to prevent real IP leaks
            if not proxy:
                proxies_env = os.getenv("RESIDENTIAL_PROXIES", "").split(",")
                valid_proxies = [p.strip() for p in proxies_env if p.strip()]
                if valid_proxies:
                    proxy = valid_proxies[0]
                else:
                    proxy = "http://jobhunt-stub-proxy:8080"

            # Using Camoufox which intercepts and overrides WebGL/Canvas natively
            async with AsyncCamoufox(headless=True, proxy=proxy) as browser:
```

### B. Changes to `scrapers/stealth_ingest.py`
We should harden the parsing of `PROXY_LIST` and the `get_stabilized_proxy` function to ensure a valid fallback is always returned even when environment variables are missing or defined as empty.

#### 1. In `PROXY_LIST` definition:
**Before:**
```python
PROXY_LIST = [p.strip() for p in os.getenv("RESIDENTIAL_PROXIES", "http://jobhunt-stub-proxy:8080").split(",") if p.strip()]
```

**Proposed After:**
```python
# Enforces enterprise residential proxies, defaulting explicitly if variable is missing or evaluated as empty
env_proxies = os.getenv("RESIDENTIAL_PROXIES", "")
if env_proxies.strip():
    PROXY_LIST = [p.strip() for p in env_proxies.split(",") if p.strip()]
else:
    PROXY_LIST = ["http://jobhunt-stub-proxy:8080"]
```

#### 2. In `get_stabilized_proxy`:
Ensure it never returns an empty dictionary. Instead, default to the stub proxy.

**Before:**
```python
def get_stabilized_proxy(session_id: str = "default") -> dict:
    if not PROXY_LIST:
        return {}
```

**Proposed After:**
```python
def get_stabilized_proxy(session_id: str = "default") -> dict:
    # Ensure there is always a fallback proxy active to prevent real IP leakage
    active_list = PROXY_LIST if PROXY_LIST else ["http://jobhunt-stub-proxy:8080"]

    # If it's a single backconnect proxy (e.g., contains @ and is a gate provider)
    # we can inject session ID into username if supported
    proxy_str = active_list[0]
    
    if len(active_list) == 1 and "@" in proxy_str and ("session-" not in proxy_str):
        parsed = urllib.parse.urlsplit(proxy_str)
        if parsed.username and parsed.password:
            new_username = f"{parsed.username}-session-{session_id}"
            port_part = f":{parsed.port}" if parsed.port else ""
            netloc = f"{new_username}:{parsed.password}@{parsed.hostname}{port_part}"
            reconstructed = urllib.parse.urlunsplit((parsed.scheme, netloc, parsed.path, parsed.query, parsed.fragment))
            return {"http": reconstructed, "https": reconstructed}

    # Otherwise, pin to a specific proxy in the list using hash of session_id
    idx = hash(session_id) % len(active_list)
    selected_proxy = active_list[idx]
    return {"http": selected_proxy, "https": selected_proxy}
```

## 5. Test Suite Review & Testing Strategy

We analyzed `tests/test_stealth_parser_and_fallbacks.py` and `tests/e2e/test_r3_scraper.py`:

### Existing Coverage
- `test_parse_page_content_multi_card_indeed`, `_linkedin`, `_fallback_to_single`, `_empty`, `_json_ld`: Adequately cover parse output structures ensuring keys are present.
- `test_process_single_job_waf_fallback`: Verifies fallback progression (direct HTTP -> `NodriverFallback` -> `ApexCamoufoxFallback`).
- `test_process_single_job_llm_fallback`: Verifies LLM parser fallback formatting.
- `test_nodriver_fallback_passes_proxy`: Specifically mocks `nodriver.start` to assert that it is initialized with `--proxy-server` matching the parameter passed.
- `test_r3_t1_get_random_proxy_fallback` in E2E tests: Verifies that proxy extraction does not crash and returns a dictionary.

### Gaps & Recommended New Tests
While existing tests cover high-level pathways, they do **not** verify the hardened protection against empty environments or missing parameters. We recommend introducing two new tests:

#### 1. In `tests/test_stealth_parser_and_fallbacks.py`:
Add a test verifying that calling the fallbacks with `proxy=None` forces loading from environment variables or defaults to the stub proxy:
```python
@pytest.mark.asyncio
@patch("nodriver.start", new_callable=AsyncMock)
async def test_nodriver_fallback_resolves_default_proxy_when_none(mock_uc_start, monkeypatch):
    from core.stealth import NodriverFallback
    # Clear environment variables to force default stub proxy
    monkeypatch.setenv("RESIDENTIAL_PROXIES", "")
    
    mock_browser = MagicMock()
    mock_page = MagicMock()
    mock_page.get_content = AsyncMock(return_value="<html></html>")
    mock_browser.get = AsyncMock(return_value=mock_page)
    mock_uc_start.return_value = mock_browser

    await NodriverFallback.get_page_content("https://example.com", proxy=None)
    mock_uc_start.assert_called_once_with(
        headless=True,
        browser_args=["--proxy-server=http://jobhunt-stub-proxy:8080"]
    )
```

#### 2. In `tests/e2e/test_r3_scraper.py`:
Add a test asserting that if `RESIDENTIAL_PROXIES` is set to an empty string, the scraper continues to yield a dictionary with the stub proxy instead of `{}` (avoiding unproxied routing):
```python
def test_get_stabilized_proxy_empty_env_fallback(monkeypatch):
    from scrapers.stealth_ingest import get_stabilized_proxy
    # Simulate empty environment setting
    monkeypatch.setenv("RESIDENTIAL_PROXIES", "")
    
    # Reload proxy module settings if necessary or test custom flow
    # Asserting that returning dictionary has stub proxy
    proxies = get_stabilized_proxy("test-sess")
    assert proxies.get("http") == "http://jobhunt-stub-proxy:8080"
```
