## 📋 Deployment Instructions

### Deploy to Render (Recommended)
```bash
# 1. Connect GitHub repo to Render dashboard
# 2. Set environment variables (see .env.example)
# 3. Render auto-deploys from render.yaml
# 4. Backend available at: https://jobhuntpro.render.com
```

### Deploy Locally (Development)
```bash
# 1. Start services
docker-compose -f docker-compose.dev.yml up -d

# 2. Run tests
python -m pytest tests/ -v

# 3. Start backend
python start_cloud.py

# 4. Start frontend (separate terminal)
cd frontend && npm run dev
```

### Deploy with Docker
```bash
# 1. Build images
docker build -t jobhunt-backend:latest .
docker build -f Dockerfile.frontend -t jobhunt-frontend:latest .

# 2. Push to registry
docker tag jobhunt-backend:latest yourregistry/jobhunt-backend:latest
docker push yourregistry/jobhunt-backend:latest

# 3. Deploy (e.g., to Docker Swarm or Kubernetes)
docker stack deploy -c docker-compose.yml jobhunt  # Swarm
# OR
kubectl apply -f k8s/ # Kubernetes
```

See [DEPLOY.md](DEPLOY.md) for complete zero-cost deployment guide.
